# SSME - Company Website Redesign

## Project Overview

This project is a complete redesign and modernization of a company website, re-implemented as a responsive and performant React application. The primary goal of this internship project was to enhance the user experience, improve the site's visual appeal, and create a more maintainable and scalable codebase. This new version is built with a modern frontend stack, focusing on delivering a seamless and engaging user journey.

The redesigned site features a clean and intuitive user interface, with clear navigation and a focus on readability. It includes several key sections: a dynamic homepage, an informative "About Us" page, a detailed "Services" section, and an easy-to-use contact form. The project emphasizes responsive design, ensuring a consistent and optimal experience across all devices, from desktops to mobile phones.

## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **Deployment:** Vercel

## Key Frontend Contributions

-   **Component-Based Architecture:** Developed a library of reusable React components to ensure consistency and streamline future development.
-   **Responsive Design:** Implemented a mobile-first approach using Tailwind CSS to create a fully responsive layout that adapts to various screen sizes.
-   **Improved UI/UX:** Designed and implemented a modern and intuitive user interface, focusing on user experience and accessibility.
-   **State Management:** Utilized React Hooks for efficient state management within functional components.
-   **Performance Optimization:** Leveraged Vite for a fast and efficient development workflow and optimized the production build for faster load times.

## Deployment

A live demo of this project is hosted on Vercel. You can view the live site at [ssme-khaki.vercel.app](https://ssme-khaki.vercel.app/).

## Disclaimer

This project is a frontend-only implementation and does not include a backend or database. The contact form is a functional UI component but does not send data to a server.

## Run locally

Install dependencies and run the dev server:

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.
